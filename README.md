# simple-storage-solidity
Basic Solidity contract to store grade and student info (struct + array)
- State variables
- Structs
- Arrays
- Mapping

## Features

- Store and retrieve a grade
- Add student information (name + marks)
- Retrieve student informatin (name + marks) using index 
- Retrieve marks using name

## Functions

- addGrade(string)
- addStudentInfo(string, uint256)
